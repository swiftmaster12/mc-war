# mc-war-plugin

A Bukkit/Paper plugin implementing a best-of-3 siege-point war system for Minecraft servers.

## Features

- Declare wars between land-owning factions via a `/war` command
- Admin-defined capture points fought over in timed battle windows
- Beacon-based visual feedback during active sieges
- Vassal/ally system with tax mechanics
- Pre-battle staging, replay summaries, and tiered campaign support
- Integrates with the [Lands](https://www.spigotmc.org/resources/lands.53313/) plugin

## Requirements

- Paper 1.20+
- [Lands](https://www.spigotmc.org/resources/lands.53313/)

## Building

```bash
mvn clean package
```
